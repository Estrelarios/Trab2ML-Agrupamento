# Busca exaustiva de parâmetros para o KMeans

Esse diretorio tem código para fazer um busca de hiperparâmetros do KMeans e busca de valores dos pesos função de avaliação que resulte em combinações de clusters diferentes (código *busca_exaustiva_pesos.py*). As imagens em resultados mostram resultados únicos e os valores para formá-los. O código *mostrar_animação.py* faz uma animação desses resultados para visualização rápida.
